# Smart Service Platform - Frontend

React-based UI for AI-Powered Service Request Platform

## Setup

1. Install dependencies: `npm install`
2. Create `.env` file from `.env.example`
3. Set API URL: `REACT_APP_API_URL=http://localhost:5000/api`
4. Run: `npm start`

## Features

- ✅ Dashboard with real-time analytics
- ✅ Service request creation & management
- ✅ AI-powered chat assistant for natural language requests
- ✅ Map-based location tracking
- ✅ Real-time status updates
- ✅ SLA tracking and alerts
- ✅ Request filtering & search
- ✅ Responsive design

## Pages

- **Dashboard**: Analytics and overview
- **Requests**: List and manage requests
- **Create Request**: Form with AI suggestions
- **AI Chat**: Natural language to structured request conversion
- **Map View**: Geo-spatial visualization of requests
